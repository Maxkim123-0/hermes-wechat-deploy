## 1️⃣ 服务器选购

### 推荐配置

| 项目 | 配置 | 说明 |
|------|------|------|
| **机型** | ECS 共享型 | ecs.e-c1m2.large |
| **CPU** | 2 核 | Hermes + Gateway 够用 |
| **内存** | 4 GB | 推荐（2GB也能跑但余量小） |
| **系统盘** | 40 GB ESSD | 系统 + Hermes + 缓存 |
| **操作系统** | Ubuntu 22.04 | 最稳，DeepSeek 官方推荐 |
| **带宽** | 1-3 Mbps | 不需要高带宽，API调用走外网 |
| **地域** | 上海/杭州 | 离 DeepSeek API 近，延迟低 |

### 💰 费用估算

| 方案 | 月费 | 说明 |
|------|------|------|
| 按量付费 | ~¥0.15/小时 ≈ ¥108/月 | 灵活，随时释放 |
| 包月（ecs.e-c1m2.large） | **¥68/月** | 性价比最高 |
| 新用户优惠 | **¥99/年** | 阿里云经常有，先领券再买 |

> **最低配置可降级：** ecs.e-c1m1.large（2核2G，~¥50/月），但内存紧，不推荐。

### 购买步骤（阿里云）

1. 打开 [aliyun.com](https://www.aliyun.com) → 登录（支付宝/淘宝账号）
2. 搜索「ECS」→ 点击「云服务器 ECS」
3. **先领新人优惠券**（首页弹窗/「最新活动」）→ 新用户有 ¥99/年活动
4. 点击「创建实例」，按上表选配置
5. **关键设置：**
   - 镜像：Ubuntu 22.04 64位
   - 登录凭证：**设置 root 密码**（牢记！）
   - 安全组：放行 22 端口（SSH）
   - 实例名称：随便填
6. 确认下单 → 支付 → 等 1-2 分钟创建完成

## 2️⃣ 连接服务器

### Windows 用户（推荐 FinalShell）

下载 FinalShell（免费）：[hostbuf.com](https://www.hostbuf.com/)

- 主机：ECS 公网 IP（阿里云控制台 → 实例详情 → 公网IP）
- 端口：22
- 用户名：root
- 密码：你刚设置的密码

### Mac / Linux 用户（终端）

```bash
ssh root@你的公网IP
# 输入密码
```

## 3️⃣ 上传部署包

### 方法一：FinalShell 拖拽上传（最简单）

FinalShell 连接后，左边文件管理 → 把 `hermes-wechat-deploy/` 整个文件夹拖到 `/root/` 目录下。

### 方法二：scp 命令行

```bash
# 先在本机把部署包打成 zip（如果还没打）
cd /path/to/hermes-wechat-deploy
zip -r hermes-deploy.zip . -x ".git/*"

# 上传
scp hermes-deploy.zip root@你的公网IP:/root/

# SSH 进去解压
ssh root@你的公网IP
cd /root
unzip hermes-deploy.zip -d hermes-wechat-deploy
```

## 6️⃣ 一键安装

```bash
cd /root/hermes-wechat-deploy

# 运行安装脚本，传入 profile 名字（建议用你的名字拼音）
bash install.sh xiaoli
```

安装脚本会自动：
1. ✅ 检查 Python 版本
2. ✅ 安装 Hermes Agent
3. ✅ 提示你输入 DeepSeek API Key
4. ✅ 创建独立 Profile
5. ✅ 复制 101 个技能
6. ✅ 生成微信二维码

**扫码步骤：**
- 脚本会生成二维码 PNG，存在 `/root/hermes/cache/images/onboard_qr.png`
- 用 FinalShell 下载这个文件到你电脑，用微信扫
- 点确认 → 脚本自动保存凭证

## 7️⃣ 启动 Gateway + 测试

```bash
# 启动 gateway（后台运行）
nohup hermes gateway run --profile xiaoli > /root/gateway.log 2>&1 &

# 等 5 秒后查看日志
tail -f /root/gateway.log
```

看到 `Gateway started` 就是成功了。

**测试：** 给你的 Bot 微信号发一条消息「你好」，Bot 会秒回。

## 8️⃣ 设为开机自启（可选）

```bash
# 启用 systemd 用户服务

# 安装 gateway 为服务

# 启动服务
hermes gateway start --profile xiaoli

# 查看状态
hermes gateway status --profile xiaoli
```

## ⚠️ 常见问题

### Gateway 启动失败？
```bash
# 看详细错误
cat /root/gateway.log
# 常见原因：API Key 无效 / 余额不足
curl -s https://api.deepseek.com/user/balance -H "Authorization: Bearer 你的Key"
```

### 发消息没反应？
- Gateway 重启后，**必须先由你发一条消息给 Bot**，Bot 才能回复
- 检查 gateway 进程：`ps aux | grep hermes | grep gateway`

### 二维码过期？
35秒自动刷新，脚本会一直轮询。

### 技能不生效？
```bash
# 确认技能已复制
ls ~/.hermes/profiles/xiaoli/skills/
# 重启 gateway
hermes gateway restart --profile xiaoli
```
