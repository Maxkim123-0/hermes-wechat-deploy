# Summarize AI — 智能摘要

**一句话：把长内容变短，保留核心信息。**

---

## 使用场景

| 场景 | 输入 | 输出格式 |
|------|------|---------|
| 网页文章 | URL | 3 要点 + 1 句话总结 |
| 聊天记录 | 对话文本 | 关键决策 + 待办事项 |
| 技术文档 | README/Doc | 核心功能 + 安装步骤 |
| 会议纪要 | 会议记录 | 决议 + 责任人 + 截止日期 |
| 视频字幕 | 字幕文本 | 核心观点 + 时间戳 |

---

## 摘要模板

### 默认格式（3-2-1）

```
📌 一句话：...
🔑 三个要点：
  1. ...
  2. ...
  3. ...
⚡ 行动项：
  - [ ] ...
```

### 技术文档格式

```
📦 项目：...
🎯 解决的问题：...
🚀 快速开始：...
⚠️ 注意事项：...
```

---

## 调用方式

1. **直接对话**：发 URL 或文本给我，说"总结"
2. **与搜索联动**：`tavily-search` 找到文章 → 自动调此技能摘要
3. **批量处理**：多个 URL 逐个摘要

---

## 与 browser 工具配合

```python
# 打开网页 → 提取内容 → 摘要
browser_navigate(url)
content = browser_console('document.body.innerText')
# 然后调此技能提炼
```

---

## 限制

- 不处理图片/视频内容（需 vision 辅助）
- 长文本（>50K 字）分块处理
- 准确性取决于原文质量
